# EcoFishTank 操作说明（面向使用人员）

本设备用于采集鱼缸相关数据，并在屏幕上显示，同时在联网后定时上传到 ThingSpeak。

## 设备会做什么

- 持续采集并显示：温度（Temperature）、水位（Height）、TDS、光照（Light）
- 自动联网：优先连接已保存的 WiFi
- 无法联网时：开启配网热点（AP）供手机连接配置
- 联网成功后：按设定间隔向 ThingSpeak 上传数据（Field1~Field4）

## 屏幕上看到什么代表什么

屏幕会显示四类信息：

- **T / TDS / Lvl / L**：分别为温度、TDS、水位（cm）、光照百分比
- **WiFi:OK**：已连接到路由器（屏幕会显示设备的 IP）
- **WiFi:AP**：设备处于配网热点模式（屏幕会显示 `Cfg:192.168.4.1`）
- **TS:OK / TS:--**：
  - `TS:OK`：最近一次上传 ThingSpeak 成功
  - `TS:--`：最近一次上传失败或尚未上传（常见于未联网/刚启动）

## 正常使用流程

- 上电后等待数秒，屏幕开始显示数据
- 如果现场 WiFi 可用且设备已配过网：
  - 屏幕显示 `WiFi:OK`，设备会按设定间隔上传 ThingSpeak
- 如果现场 WiFi 不可用或尚未配网：
  - 屏幕显示 `WiFi:AP`，此时使用手机进行配网

## 如何配网（手机操作）

1. 打开手机 WiFi 列表，找到以 `EcoFishTank-` 开头的热点并连接
2. 热点密码：`ecofish123`
3. 打开手机浏览器，访问：`http://192.168.4.1`
4. 在页面中填写/选择：
   - WiFi SSID（可下拉选择或手动输入）
   - WiFi 密码
   - ThingSpeak 发送间隔（时/分/秒）
5. 点击“保存”
6. 等待片刻后点击页面“刷新”，观察状态是否显示为已连接

说明：

- **ThingSpeak 发送间隔**可填小时/分钟/秒三个格子。
- 三项都为 0 时表示使用默认间隔。

## 如何强制进入配网模式（按钮）

设备带有一个“用户按钮”。

- **无论设备当前是否已联网**，只要按下用户按钮，设备会立刻进入配网热点模式（`WiFi:AP`）。
- 适用场景：更换路由器、忘记 WiFi 密码、现场需要重新配置等。

## 常见问题与处理

- **手机搜不到 `EcoFishTank-xxxxxx` 热点**
  - 先看屏幕是否显示 `WiFi:AP`（只有在 AP 模式下才会有热点）
  - 按一下用户按钮强制进入 AP 模式
  - 让手机靠近设备、关开一次手机 WiFi 重新扫描
  - 仍不出现：断电重启设备后再试

- **屏幕显示 `WiFi:AP`，但 ThingSpeak 一直失败**
  - 这是正常现象：AP 模式只是用于配网，本身不代表已接入互联网
  - 请先用手机完成配网，让屏幕显示 `WiFi:OK` 后再观察上传状态

- **屏幕显示 `WiFi:OK`，但 ThingSpeak 仍失败**
  - 可能是现场网络限制/无外网，或 ThingSpeak Key 配置不正确
  - 先确认现场网络可正常访问互联网，然后检查 ThingSpeak 配置是否正确

- **需要重新开始一次配网**
  - 直接按用户按钮进入 AP 模式，按“如何配网”重新配置即可
